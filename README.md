# World History Korean High School

고등학교 세계사 학습을 위한 마스터맵 앱 프로젝트입니다.

## 목표

이 앱은 단순 암기장이 아니라, 학생이 세계사를 다음 네 축으로 이해하도록 돕는 것을 목표로 합니다.

1. 시간축: 어느 시대의 사건인가?
2. 지역축: 어느 지역 세계의 변화인가?
3. 인과축: 왜 일어났고 무엇이 바뀌었는가?
4. 시험축: 내신·수능형 자료에서 어떻게 판별하는가?

최종 목표는 “이 앱 하나로 고등 세계사의 전체 구조를 머릿속에 세울 수 있는 학습 도구”입니다.

## 핵심 원칙

학생 성적과 연결될 수 있는 앱이므로, 역사 콘텐츠는 무분별하게 자동 생성하지 않습니다.

- 앱 구조와 UI는 Codex가 구현합니다.
- 세계사 학습 콘텐츠는 별도 데이터 파일로 분리합니다.
- 모든 학습 데이터는 검수 상태를 표시합니다.
- 검수 전 콘텐츠는 학생용 확정 콘텐츠처럼 보이지 않게 합니다.
- 사건·연도·왕조 순서·인물·시험 포인트는 반드시 사람이 검수합니다.

## 1차 MVP

1차 MVP에서는 다음 기능만 구현합니다.

- 시대 필터
- 지역 필터
- 검색창
- 사건 카드 목록
- 선택한 사건 상세 패널
- 검수 상태 표시
- 추후 퀴즈와 약점 분석을 붙일 수 있는 구조

퀴즈, 로그인, 진도 저장, 대량 학습자료 생성은 이후 단계에서 진행합니다.

## 데이터 설계 방향

세계사 사건 데이터는 코드와 분리된 파일에서 관리합니다.

예상 위치:

```txt
/src/data/worldHistoryEvents.ts
/src/types/worldHistory.ts
```

각 사건 카드는 다음 성격을 가져야 합니다.

- 한 줄 요약
- 원인
- 결과
- 연결 사건
- 시험 포인트
- 헷갈리기 쉬운 점
- 검수 상태

## 프로젝트 역할 분담

- Codex: 앱 구조, UI, 검색·필터, 데이터 표시, 반응형 구현
- Ain/ChatGPT: 학습 구조 설계, 사건 카드 초안, 비교 구조, 퀴즈 설계
- 주원쌤: 교육자 관점의 최종 검수
